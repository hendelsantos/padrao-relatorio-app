# Como criar um ícone personalizado

## Requisitos para o ícone:
- **Tamanho recomendado**: 1024x1024 pixels (PNG com fundo transparente)
- **Formato**: PNG
- **Nome do arquivo**: `app_icon.png`
- **Local**: `assets/icon/app_icon.png`

## Sugestões de design para o app de manutenção:
1. **Tema**: Ferramentas, engrenagens, relatórios
2. **Cores**: Azul/Verde (conforme o tema do app)
3. **Estilo**: Moderno, clean, profissional

## Ferramentas recomendadas:
- **Online**: Canva, Figma, GIMP
- **IA**: Dall-E, Midjourney, Stable Diffusion
- **Desktop**: Adobe Illustrator, Photoshop

## Exemplo de prompt para IA:
"Create a modern app icon for a maintenance report application. Include a wrench or gear symbol with a clipboard or document. Use blue and green colors. Make it professional and clean. 1024x1024 pixels, transparent background."

## Após criar o ícone:
1. Salve como `app_icon.png` nesta pasta
2. Execute: `flutter pub get`
3. Execute: `dart run flutter_launcher_icons`
4. Rebuild o app
